# godot-project-base
A base for Godot projects.

## Features

- CheeGames intro
- `StateMachine` class
- `State` class
- `DamageManager` class
- `AwaitTimer` class
- `Globals` class
- `Gamemanager` singleton
- `Settings` singleton
- `Pathinding` singleton
- `Cursor` singleton
- Hurt and Hitboxes
- Loading zone
- Player detector
- Screenshake
- Flash effect
- Camera mouse weight
- Shadow generator
- Base `Entity` scene
- A topdown player controller
